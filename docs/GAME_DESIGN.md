# Game Design

## First Slice

- Show an 8 by 8 board.
- Use circular pieces as temporary art.
- Allow pieces to be dragged and dropped onto empty squares.
- Keep Mak-yek-specific move validation for a later step.

## Initial Placeholder Setup

- Dark pieces start on rows 1 and 2.
- Light pieces start on rows 7 and 8.
- Middle four rows start empty.
